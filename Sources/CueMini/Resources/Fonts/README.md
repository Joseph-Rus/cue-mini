# Fonts

Drop these `.ttf` files into this folder to upgrade the typography beyond the
SF Pro / system-monospaced fallbacks.

## Inter Tight (display + body)

Download from <https://fonts.google.com/specimen/Inter+Tight> and place:

- `InterTight-Regular.ttf`
- `InterTight-Italic.ttf`
- `InterTight-Medium.ttf`
- `InterTight-SemiBold.ttf`
- `InterTight-Bold.ttf`

## JetBrains Mono (caption)

Download from <https://www.jetbrains.com/lp/mono/> or
<https://fonts.google.com/specimen/JetBrains+Mono> and place:

- `JetBrainsMono-Medium.ttf`
- `JetBrainsMono-SemiBold.ttf`

The app registers anything in this folder at launch via `Fonts.registerBundled()`.
